# Sistema de Gestión de Estudiantes

Este proyecto implementa un sistema simple para gestionar una lista de estudiantes de una escuela, desarrollado como taller final de Programación Orientada a Objetos (POO).

## Descripción

El sistema permite:
- Registrar estudiantes con nombre, número de identificación y calificación
- Buscar estudiantes por su número de identificación
- Listar todos los estudiantes registrados
- Validar datos de entrada para garantizar información correcta

## Estructura del Proyecto

El proyecto consta de dos archivos principales:

- **Estudiante.java**: Define la clase Estudiante con sus atributos y métodos
- **Escuela.java**: Contiene la clase Escuela que gestiona una colección de estudiantes y la lógica del programa

## Requisitos

- Java JDK 8 o superior

## Instalación y Ejecución

1. Clone este repositorio o descargue los archivos fuente
2. Abra una terminal o línea de comandos
3. Navegue hasta el directorio donde se encuentran los archivos
4. Compile el código.


## Uso

Al iniciar el programa, se cargan automáticamente tres estudiantes de ejemplo y se muestra un menú con las siguientes opciones:

1. **Agregar nuevo estudiante**: Permite registrar un nuevo estudiante con validación de datos
2. **Buscar estudiante por ID**: Busca y muestra la información de un estudiante según su número de identificación
3. **Mostrar todos los estudiantes**: Lista todos los estudiantes registrados en el sistema
4. **Salir**: Termina la ejecución del programa
